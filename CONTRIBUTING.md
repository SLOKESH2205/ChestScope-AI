# Contributing to ChestScope-AI

We welcome contributions from researchers, clinicians, and developers to improve ChestScope-AI.

## Code Style
- Follow PEP 8 guidelines for Python code.
- Add descriptive docstrings and portfolio-friendly comments.
- Keep the Streamlit UI components modular under the `ui/` directory.

## Testing
- Ensure all tests pass before proposing updates:
  ```bash
  py -m pytest chest_xray_classifier/tests/
  ```

## Issues & Pull Requests
- Open detailed issues outlining bugs or suggestions.
- When submitting PRs, document changes in `CHANGELOG.md`.
