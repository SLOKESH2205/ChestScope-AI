# Security Policy

## Supported Versions
We actively maintain and support the stable releases of ChestScope-AI:

| Version | Supported |
| :--- | :---: |
| 1.2.x | Yes |
| < 1.2.0 | No |

## Reporting a Vulnerability
Please report security vulnerabilities by opening an issue or contacting the authors directly. Do not disclose vulnerabilities publicly until a patch has been published.
